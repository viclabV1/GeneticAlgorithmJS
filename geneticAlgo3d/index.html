Genetic Algorithm

GitHub
Explanation

This is a simulator of a genetic algorithm in a basic 3D environment. Each entity in the simulation has a set of "genes". The goal of every entity is to reach the red cube. After 10 seconds, the two closest entities are chosen for the creation of a new generation of entities. The genes of those closest entities are mixed using single point crossover. Over time, the overall population gets better and better at reaching the goal.

Paramaters

entityCount is the number of entities that exist in the simulation.

mutRate defines the max and min of possible mutation on each vector element in the genes.

geneCount is the number of "genes" each entity has that determines its motion.
